## 写作原则

### 1. 情节极具吸引力，悬念感拉满
剧情能牢牢抓住读者好奇心，全程赋予读者强烈期待感，反转与走向往往意料之外、情理之中，无突兀生硬的设定，每一处剧情转折都有前文铺垫支撑，让读者始终有追读欲。

### 2. 情节画面感与代入感双在线
用细腻的场景、动作、细节描写搭建出鲜活的故事画面，让读者仿佛身临其境；剧情贴合真实的生活逻辑与情感逻辑，摒弃悬浮空洞的桥段，让读者能轻松代入角色的处境与心境。

### 3. 剧情设定贴合人性，充满真实烟火气
摆脱AI式的模板化、套路化创作，情节发展、人物选择都贴合真实人类的思维与行为逻辑，有温度、有瑕疵、有下意识的反应，而非冰冷的剧情堆砌，完完全全像是真人亲历的故事。

### 4. 情感与价值观引发深度共鸣
精准刻画爱情、亲情、欲望等多元情感，不刻意煽情也不刻意冷漠，传递的价值观贴近大众的内心感受，能戳中读者的情感软肋，让读者在故事里看到自己、共情他人。

### 5. 深挖人性复杂，不做单一标签化塑造
拒绝非黑即白的人物与剧情，全面展现人性的多面性——善良与自私、勇敢与懦弱、深情与算计并存，刻画人物在情感、欲望、道德中的挣扎与抉择，还原真实的人性百态。

### 6. 剧情节奏张弛有度，阅读感舒适
节奏把控恰到好处，紧张的冲突桥段与舒缓的情感铺垫交替进行，有起有伏、松紧相宜，全程不给读者压抑、窒息的阅读感受，保证流畅舒服的阅读体验。

### 7. 表达含蓄有韵味，暗藏隐喻留白
摒弃论文式的直白说教，不生硬灌输道理，巧用隐喻、象征、留白的手法，将情感、主旨藏在情节与细节里，留足意会空间，让读者自行品味感悟，余韵悠长。

### 8. 开篇直击人心，瞬间抓住注意力
开头摒弃冗长铺垫，直接抛出悬念、冲突或极具张力的场景，第一时间抓住读者眼球，让读者一眼沦陷，产生继续读下去的强烈意愿。

### 9. 世界观与逻辑严谨自洽，无前后矛盾
世界观设定统一且闭环，剧情发展、人物行为、规则设定全程逻辑严谨，无前后相悖、漏洞百出的问题，所有情节都遵循自身的故事逻辑，经得起推敲。
